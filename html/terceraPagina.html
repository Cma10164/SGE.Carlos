Proceso de Instalación

Para instalar este SGE gratuito en tu ordenador, sigue estos pasos:

  1. Descarga el software desde su página oficial.
  2. Ejecuta el instalador y sigue las instrucciones.
  3. Configura los módulos básicos según las necesidades de tu empresa.

Administración y Configuración

Una vez instalado, puedes administrar el SGE configurando usuarios, permisos y módulos.

Acceso Seguro y Privilegios

El sistema permite configurar roles como administrador, usuario estándar y supervisor.

Generación y Exportación de Informes

Para generar informes, accede al panel de reportes y selecciona los datos deseados.
